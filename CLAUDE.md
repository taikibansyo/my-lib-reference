## プロジェクト概要

MLBトレーディングカードの日本語データベース＋コレクション管理サービス。

- **リポジトリ**: `my-lib-reference`（将来独立化予定）
- **ブランチ**: `app_mlb_db`
- **テックスタック**: Next.js 15 (App Router) / React 19 / TypeScript strict / Tailwind CSS / Radix UI
- **データ**: CSV 432件（1997〜2025年、主にTopps）→ Supabase移行予定
- **ビジョン**: 日本語圏で不足しているMLBトレカ情報基盤を作り、コレクション管理ツールとして展開

## コーディング規約

### TypeScript
- `strict: true` を維持する
- `any` は使わない。必要な場合は `unknown` + 型ガード
- 型定義は `mlb_card_db/lib/` 配下に集約
- フィールド名は既存の `MlbCardRecord` 型に準拠

### コンポーネント
- Server Components をデフォルトとし、インタラクションが必要な箇所のみ `"use client"`
- コンポーネントは `mlb_card_db/components/` に配置
- Radix UI + Tailwind CSS でスタイリング（インラインスタイル禁止）

### ファイル構成
- コアロジック: `mlb_card_db/lib/`
- UIコンポーネント: `mlb_card_db/components/`
- データ: `mlb_card_db/data/`
- ページ: `src/app/`
- スクリプト: `scripts/`

### テスト
- Vitest + @testing-library/react
- 新機能には必ずテストをつける
- `npm run test` が通る状態を維持

### Git
- ブランチ: `app_mlb_db` をメインとして使用
- コミットメッセージ: `feat:` / `fix:` / `refactor:` / `docs:` / `test:` プレフィックス
- 日本語でのコミットメッセージOK
